# 專案文件

此資料夾存放專案相關的文件與規劃資料。

## 📂 資料夾內容

### 方案A_Excel資料/
專案完整規劃的 Excel 資料（CSV 格式），包含 14 個檔案：

1. **01_專案概述.csv** - 專案基本資訊
2. **02_技術堆疊.csv** - 所有技術選型與理由
3. **03_訓練資料配比.csv** - 訓練資料來源配比
4. **04_標籤定義.csv** - 意圖分類標籤定義
5. **05_模型訓練參數.csv** - BERT 訓練參數詳細說明
6. **06_效能評估指標.csv** - 準確率、F1 Score 等指標
7. **07_混淆矩陣.csv** - 混淆矩陣數據
8. **08_系統效能.csv** - 推論時間、記憶體占用等
9. **09_方案比較.csv** - 與其他方案的比較
10. **10_簡報評分對應.csv** - 簡報評分項目對應
11. **11_Word評分對應.csv** - Word 報告評分項目對應
12. **12_三週時程規劃.csv** - 詳細開發時程表
13. **13_套件清單.csv** - Python 套件清單
14. **14_核心功能清單.csv** - 功能模組列表

### 使用方式

- 每個 CSV 檔案可以直接用 Excel 開啟
- 或在 Excel 中合併成一個工作簿
- 適合用於專題簡報和報告撰寫
